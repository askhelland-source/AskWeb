HTTP og HTTPS hva er det

Hva er HTTP?

HTTP er protokoll for å sende data på interenett. Om du klikker på knapper eller laster opp bilder så er det HTTP som sørger for at dataen når fram.

Interennett bruker HTTPS hvordan

Hvordan bruker vi HTTPS?

Vi bruker HTTPS som sikringsprotokoll på nettet. Du som bruker sender mange forespørsler. Disse forespørslene blir kryptert av når du bruker HTTPS.

HTTP eller HTTPS

Hvorfor HTTPS?

Vi ønsker å bruke HTTPS for personvern på nett. Noen ganger kan vi få melding om at siden ikke er sikker, men at du kan velge å gå videre. Dette er ofte fordi de kun bruker HTTP og ikke HTTPS.
