Hva er react?

React er et stort JavaScripts bibliotek for å bygge nettisder. React lar deg gjennburke komponenter som du kan sette sammen.

På innsiden av React

Hvordan brukes React?

React brukes i dag av Meta og små og store bedrifter. React brukes til å bygge nettisder og mobilapper gjennom react native.

Gjennbruk komponenter

Hvorfor React?

React oppdaterer kun de nødvendige delene av siden av siden som er endret. Dette gir bedre ytelse. Du lager komponeter som du kan gjennbruke mange steder. Tilegg så er det veldig populært derfor er det lett å finne hjelp og insiparsjon.
