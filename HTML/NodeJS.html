JavaScript og NodeJS?

Hva er NodeJS?

NodeJS er en måte for å kjøre JavaScript på en server. Normalt brukes JavaScript kunn i netteleseren. Med nodeJS får vi muligheten til å kjøre JavaScript i en server.

En server på din PC

Hvordan brukes NodeJS?

NodeJS brukes i dag til å lage Apier, servere, automatisere oppgaver og mer. For å starte med NodeJS trenger må du ha innstallert NodeJS på din datamskin. Deretter kan vi lage JavaScript kode for å kunne drive med backend av en server.

NodeJS mestring

Hvorfor NodeJS?

Grunnen til at vi bruker NodeJS er fordi det er et veldig populært løsning. Tilegg så er det veldig lettere å lære et språk som kan gjelde både for frontend og backend.
